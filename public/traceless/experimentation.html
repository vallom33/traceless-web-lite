Pourquoi ce modèle ?

Sur le web moderne, les images se chargent automatiquement, souvent avant même que l'utilisateur ne scrolle jusqu'à elles. Cela représente :

  • Des données consommées inutilement
  • De la bande passante gaspillée
  • De l'énergie dépensée (serveurs, réseau, appareil)
  • Du temps d'attention capturé sans consentement

En rendant le chargement explicite, nous redonnons le contrôle à l'utilisateur. Le texte suffit souvent à comprendre le contenu.

Sans JavaScript

Si JavaScript est désactivé dans votre navigateur, les boutons ci-dessus ne fonctionneront pas. Mais ce n'est pas grave : les descriptions textuelles vous donnent toutes les informations essentielles.

Le contenu reste accessible sans images et sans JavaScript.
